## 情報工学科
*コンピューターの基礎から高度なプログラミングまで*
*情報化社会をさらに進化させる、IT業界の新たなリーダーを育成*
**3つのコース**
1. システムエンジニアコース
- プログラミングによって時代の求める新しい情報システムを作る
2. インターネットアンドセキュリティコース
- 安全で安心できる情報社会を作る
3. 知能情報コース
- 〈もの〉と〈こと〉の未来へAIの先を築く
